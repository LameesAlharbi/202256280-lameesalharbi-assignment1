# Personal Portfolio Website

This project is a responsive personal portfolio website developed as part of SWE363 – Assignment 1.  
It presents my background, selected projects, and contact information in a clean and structured layout.

The website serves as the foundation of my professional portfolio and will be expanded in future assignments.

---

## Features

- About Me section with personal introduction
- Projects section showcasing academic work
- Contact form (frontend only)
- Dark / light mode toggle
- Greeting message based on time of day
- Smooth scrolling navigation
- Responsive layout for desktop, tablet, and mobile
- Clickable social media icons linking to LinkedIn and GitHub

---

## Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (DOM manipulation and interactivity)

---

## How to Run Locally

1. Download or clone the repository
2. Open the project folder
3. Open `index.html` in your browser

No installation required.
